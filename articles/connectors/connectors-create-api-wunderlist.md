# <a name="monitor-and-manage-wunderlist-in-azure-logic-apps"></a>Supervisión y administración de Wunderlist en Azure Logic Apps

A partir del 6 de mayo de 2020, el servicio Wunderlist se cerró y solicitó a los clientes que migraran sus datos y empezaran a usar Microsoft To Do en su lugar. Puede usar el [conector de Microsoft To Do](/connectors/todo/) para reemplazar el [conector de Wunderlist](https://preview.flow.microsoft.com/connectors/shared_wunderlist/wunderlist/).
