- **[Supervisión con paneles de Azure Data Studio](../articles/azure-arc/data/azure-data-studio-dashboards.md)**
> Azure Data Studio es una herramienta de base de datos multiplataforma para profesionales de datos que usan la familia de las plataformas de datos locales y en la nube de Microsoft en Windows, macOS y Linux. Azure Data Studio ofrece una experiencia de editor moderna con IntelliSense, fragmentos de código, integración del control de código fuente y un terminal integrado. Se ha diseñado para usuarios de plataformas de datos, con gráficos integrados de conjuntos de resultados de consultas y paneles personalizables.

**[Supervisión con Grafana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
> Grafana es una aplicación web multiplataforma de código abierto para la visualización interactiva y el análisis. Proporciona gráficos, grafos y alertas para la web cuando se conecta a orígenes de datos admitidos. Es expandible mediante un sistema de complementos. Los usuarios finales pueden crear paneles de supervisión complejos mediante la consulta interactiva.

**[Log Analytics con Kibana](../articles/azure-arc/data/monitor-grafana-kibana.md)**
>Kibana es un panel de visualización de datos de código abierto para Elasticsearch. Proporciona funcionalidades de visualización basadas en el contenido indexado en un clúster de Elasticsearch. Los usuarios pueden crear diagramas de barras, de líneas y de dispersión, o gráficos circulares y mapas basados en grandes volúmenes de datos.
